YADS — YummyAnime Discord Status

Это приложение отображает вашу активность на сайте YummyAnime в Discord.

Как пользоваться:

1. Распакуйте архив

2. Запустите приложение:

   - YADS.exe — запуск с открытым окном командной строки  
     (приложение работает, пока окно открыто)

   - start.vbs — скрытый запуск (рекомендуется)  
     приложение работает в фоне без окна

Готово! Теперь в Discord видно, когда вы смотрите аниме на YummyAnime.

Примечания:
При скрытом запуске приложение сразу запускается в фоне без каких-либо появлений командной строки и т.п. Его можно увидеть и остановить через диспетчер задач (процесс Node.js).
Также рекомендую закинуть ярлык start.vbs в папку автозагрузки Windows, чтобы всё работало само и не надо было каждый раз что-то включать
